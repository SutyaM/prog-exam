linux

1. IEEE altal meghozott standard ami lehetove teszi az op. rendszerek kozotti kompatibilitast

2. protokol ami lehetove teszi a grafikus felulet hasznalatat

3. stdin = bemenet (pl: billentyuzet), stdou = kimenet (pl: monitor)

4. hatterben futo programok

5. /bin - olyan futtathato programok amik elengedhetetlenek a rendszer mukodesehez
   /etc - rendszerbeallitasok
   /usr - olyan programok amiket a felhasznalo hasznal (nem elengedhetetlenek)
   /home - felhasznalok mappai/fajljai

6. rejtett fajlok

7. relativ: ./dir/file.txt
   abszolut: /home/username/dir2/file3.txt
